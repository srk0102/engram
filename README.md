# Engram

Your API learns to defend itself.

Every request leaves a trace.
Known patterns handled in 0.1ms.
Novel patterns wake the brain once.
Brain silent after learning.
Cost: $0 after session 3.

## Install in Supabase

Paste supabase/install.sql into
your SQL editor. Run it. Done.

Patterns appear in your dashboard.
No new infrastructure.
No code changes.
Uses your existing Postgres.
